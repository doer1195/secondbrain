# 문제 주소
https://cses.fi/problemset/task/2216

# 문제
1..n까지의 순열이 순서가 무작위로 주어졌을 때, 1..n까지의 수를 순서대로 모아보자. 한번에 한번만 좌에서 우로 순회할 수 있다. 몇번 순회해야 하는가?

# 해결


1..n까지의 수를 순서대로 모아야한다.  
  

$f(x) = x의 \ 인덱스$ 라고 할 때, $f(x)$ 기준으로 정렬한 후 $f(x+1) < f(x)$ 가 되는 지점이 몇 개 인지 확인하면 된다.
